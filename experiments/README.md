# Experiments

This directory is reserved for future reproducible experiments. Volume I has no
separate experiments: its deterministic demonstrations live in the source
package and are invoked through the CLI.

Do not place exploratory scripts here merely to bypass the chapter progression.
Any future experiment must state its question, fixed inputs, exact command,
expected output location, and interpretation; generated measurements must never
be fabricated or silently committed as textbook evidence.

Return to the [documentation index](../docs/README.md), read the project's
[design philosophy](../docs/design-philosophy.md), or consult the
[contributor guide](../CONTRIBUTING.md) before proposing experiment material.
